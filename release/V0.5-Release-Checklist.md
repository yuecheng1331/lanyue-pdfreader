# LocalPdfReader V0.5 发布清单

## 1. 自动发布门禁

运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\Publish-WindowsX64.ps1
```

脚本必须全部完成：

- 还原依赖并运行 Release 全量测试；
- 发布 Windows 11 x64 自包含主程序与 PDF Worker；
- 检查 `pdfium.dll`、`e_sqlite3.dll` 和 .NET 自包含运行时；
- 拒绝数据库、设置、凭据、日志、密钥和调试符号进入发布包；
- 生成 `LocalPdfReader-0.5.0-win-x64.zip`；
- 生成同名 `.sha256` 文件和 `release-manifest.json`；
- 解压 ZIP 并逐文件核对大小与 SHA-256；
- 在禁用系统 .NET 运行时查找且使用隔离用户数据目录时启动主程序和 PDF Worker。

## 2. 产物检查

- ZIP 文件名和关于页面均显示 `0.5.0`；
- `release-manifest.json` 中 `tests` 为 `passed`；
- `extractedArchiveVerified` 为 `true`；
- `isolatedLaunchVerified` 为 `true`；
- `.sha256` 中的校验值与重新计算结果一致；
- ZIP 不包含 `reader.db`、`settings.json`、备份设置、日志、API 密钥或凭据；
- ZIP 解压后不需要安装 .NET 或复制其他运行库。

## 3. Windows 11 x64 人工验收

- 从全新目录解压 ZIP 并启动 `LocalPdfReader.App.exe`；
- 启动时显示最近文件主页，不自动打开旧 PDF；
- 打开多个 PDF，验证标签切换、排序和关闭；
- 验证翻页、缩放、适合页面、适合宽度和旋转；
- 验证文字选择、复制和现有 DeepSeek 配置升级后的翻译；
- 验证全文搜索、取消、结果跳转和高亮；
- 验证四种批注颜色、笔记、定位、删除和重启恢复；
- 结束 PDF Worker，确认多标签、阅读状态和批注能够恢复；
- 检查关于页面中的版本、提交、PDFium 和 .NET 信息；
- 确认用户数据只写入 `%LocalAppData%\LocalPdfReader`，不写入解压目录。

## 4. 正式发布收尾

人工验收通过后：

1. 提交阶段 F 源码、测试、脚本和发布清单；
2. 确认构建、测试和发布输入无未提交改动，再重新运行完整发布脚本；
3. 确认清单中的 `dirtyBuild` 为 `false`；
4. 创建带说明的 `v0.5.0` 标签；
5. 保留 ZIP、`.sha256` 和 `release-manifest.json` 作为正式发布产物；
6. 不自动推送提交、标签或发布产物。
