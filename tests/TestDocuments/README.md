# V0.1 固定测试文档集

G05 使用 `PdfTestDocumentFactory` 在测试临时目录中确定性生成以下 PDF，避免把来源不明或有版权的论文文件提交到仓库：

| 场景 | 自动验证内容 |
| --- | --- |
| 中文及空格路径 | Worker 能打开、渲染并提取文本 |
| A4 纵向页面 | 页面尺寸、BGRA32 位图和共享内存释放 |
| 横向页面 | 原始宽高与渲染尺寸 |
| 三页文档 | 页数及每页独立文本 |
| 多行英文与断词 | 原始文字和字符坐标可提取 |
| 无文字层页面 | `HasTextLayer=false`，仍可渲染 |
| 只读 PDF | 不要求写权限即可打开和渲染 |
| 损坏 PDF | 返回明确错误，Worker 与主程序不崩溃 |
| 四方向旋转 | 90°/270° 正确交换目标位图宽高 |
| Worker 强制终止 | 断开检测、重新启动和重新握手 |

自动测试不会调用 DeepSeek，不读取 Windows 中保存的 API 密钥，也不会产生 API 费用。

仍需人工文档验证的场景：真实中文嵌入字体论文、双栏论文、公式与引用附近选区、高分辨率图片、透明图层、高 DPI 显示器。密码 PDF 当前不属于已经实现的打开流程，不能标记为通过。
